# Nordic EV Adoption & Infrastructure Analysis (2020-2025)

## 📌 Project Overview
As Sweden and Norway lead the global transition to electric mobility, this project explores the relationship between EV sales and the rollout of public charging infrastructure. The analysis focuses on identifying "Infrastructure Deserts" in Northern Europe.

## 🛠️ Technical Stack
* **Language:** Python (Jupyter Notebook)
* **Libraries:** Pandas (Data Manipulation), Matplotlib/Seaborn (Visualization).
* **Data Source:** IEA Global EV Outlook 2025 & EAFO.

## ❓ Business Questions Answered
1. Which Nordic country has the best "Car-to-Charger" ratio?
2. Has the growth of Fast Chargers (DC) kept pace with Battery Electric Vehicle (BEV) sales?
3. What is the projected charger requirement for Gothenburg by 2027?
